-- Directory structure                                   --

- bitvis_vip_axistream: A VIP containing VVC and BFM
  - doc               : Quick Reference for BFM and VVC
  - script            : Compile scripts
  - src               : AXI4-Stream BFM and VVC source code

IMPORTANT
This is a simplified Verification IP (VIP) for AXI4-Stream.
The given VIP complies with the basic AXI4-Stream protocol and thus allows a 
normal access towards a AXI4-Stream interface. This VIP is not a AXI4-Stream protocol checker.
For a more advanced VIP please contact Bitvis AS at support@bitvis.no
