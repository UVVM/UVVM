IMPORTANT
This is a Verification IP (VIP) for Avalon MM.
The given VIP complies with the Avalon-MM protocol and thus allows a 
normal access towards an Avalon-MM interface. This VIP is not an Avalon-MM protocol checker.
For a more advanced VIP please contact Bitvis AS at support@bitvis.no
